## 📋 Sobre o Evento

O **Coders.jar Tech Summit BS 2026** é uma excelente oportunidade para aprendizado, networking e troca de experiências sobre o universo Java. Este evento reúne profissionais e entusiastas da tecnologia para discutir as tendências mais atuais do ecossistema Java, incluindo IA, arquitetura de software e computação serverless.

### 🎯 Público-Alvo

- 👨‍💻 Programadores e Programadoras
- 🏗️ Engenheiros e Engenheiras de Software
- 🎓 Cientistas da Computação
- 📚 Estudantes de áreas relacionadas à tecnologia

### 13:30 - 14:10 | 1ª Palestra
**Virtual Threads + LangChain4j: Escalabilidade e Performance em Sistemas Inteligentes**

**Palestrante:** Elder Moraes  
*Java Champion | Principal Developer Advocate na IBM*  
[LinkedIn](https://www.linkedin.com/in/eldermoraes/)

**Descrição:** Aplicações Java que integram IA enfrentam problemas de latência. Descubra como Virtual Threads e LangChain4j podem resolver esses desafios, permitindo processar múltiplas requisições sem bloqueio e construindo sistemas inteligentes de alto desempenho.

**Nível:** Intermediário/Avançado

### 14:10 - 14:50 | 2ª Palestra
